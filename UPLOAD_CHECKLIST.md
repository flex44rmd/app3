# ✅ قائمة التحقق من رفع الملفات إلى InfinityFree

## قبل الرفع

- [ ] تم إنشاء مجلد `htdocs` محلياً
- [ ] تم نسخ جميع الملفات إلى مجلد `htdocs` (استخدم `copy-to-htdocs.ps1`)
- [ ] ملف `index.html` موجود في `htdocs/`
- [ ] ملف `.htaccess` موجود في `htdocs/`

## الملفات المطلوب رفعها

### ملفات أساسية
- [ ] `index.html`
- [ ] `main.dart.js`
- [ ] `flutter.js`
- [ ] `flutter_bootstrap.js`
- [ ] `flutter_service_worker.js`
- [ ] `favicon.png`
- [ ] `manifest.json`
- [ ] `version.json`
- [ ] `.htaccess`

### مجلدات
- [ ] `assets/` (مع جميع المحتويات)
- [ ] `canvaskit/` (مع جميع المحتويات)
- [ ] `icons/` (مع جميع المحتويات)

## خطوات الرفع

### 1. إنشاء المجلد على الخادم

- [ ] سجل الدخول إلى لوحة تحكم InfinityFree
- [ ] اذهب إلى **File Manager**
- [ ] أنشئ مجلد باسم `htdocs` (أو الاسم المطلوب حسب إعدادات النطاق)
- [ ] تأكد من أن اسم المجلد يطابق الإعدادات في قسم **Domains**

### 2. رفع الملفات

#### الطريقة الأولى: File Manager
- [ ] افتح مجلد `htdocs` في File Manager
- [ ] ارفع جميع الملفات والمجلدات من `htdocs/` المحلي
- [ ] تأكد من رفع جميع المجلدات الفرعية

#### الطريقة الثانية: FTP
- [ ] احصل على معلومات FTP من لوحة التحكم
- [ ] استخدم FileZilla أو أي عميل FTP
- [ ] اتصل بالخادم
- [ ] ارفع جميع الملفات إلى المجلد الصحيح

### 3. التحقق من الصلاحيات

- [ ] تأكد من أن ملفات `.htaccess` و `index.html` لديهم صلاحيات القراءة
- [ ] تأكد من أن جميع المجلدات لديها صلاحيات التنفيذ

### 4. إعداد قاعدة البيانات

- [ ] سجل الدخول إلى phpMyAdmin
- [ ] اختر قاعدة البيانات الخاصة بك
- [ ] نفّذ ملف `sql/schema.sql` لإنشاء الجداول

### 5. التحقق من النجاح

- [ ] افتح المتصفح واذهب إلى: `https://al3adrar.great-site.net`
- [ ] تحقق من أن الموقع يعمل
- [ ] تحقق من أن جميع الملفات يتم تحميلها (افتح Developer Tools → Network)
- [ ] تحقق من عدم وجود أخطاء في Console

## استكشاف الأخطاء

### خطأ: "Directory is missing"
- [ ] تحقق من إنشاء المجلد في المكان الصحيح
- [ ] تحقق من إعدادات النطاق في لوحة التحكم
- [ ] تأكد من أن اسم المجلد يطابق الإعدادات

### خطأ: "404 Not Found"
- [ ] تحقق من وجود `index.html` في المجلد الصحيح
- [ ] تحقق من ملف `.htaccess`
- [ ] تحقق من مسارات الملفات

### خطأ: "Files not loading"
- [ ] تحقق من مسارات الملفات في `index.html`
- [ ] تأكد من رفع جميع المجلدات
- [ ] تحقق من صلاحيات الملفات

### خطأ: "CORS Error"
- [ ] تحقق من إعدادات CORS في `.htaccess`
- [ ] إذا كنت تستخدم API خارجي، تأكد من إعدادات CORS في الخادم

## ملاحظات مهمة

1. **اسم المجلد:** يجب أن يطابق الإعدادات في لوحة تحكم InfinityFree
2. **API Backend:** InfinityFree لا يدعم Node.js - ستحتاج إلى:
   - استخدام PHP كبديل (سيتم إنشاء ملفات PHP)
   - أو استخدام خدمة خارجية لـ Node.js (Render, Railway, Heroku)
3. **قاعدة البيانات:** تأكد من أن قاعدة البيانات نشطة
4. **الأمان:** لا ترفع ملف `.env` إلى الخادم
